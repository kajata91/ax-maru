# AX-Maru — 1단계 MVP 실행 가이드

> 이 문서 하나로 오늘 바로 AX-Maru를 돌려볼 수 있습니다.
> 구성: (A) Obsidian 폴더 구조 → (B) Cowork에 붙여 넣을 지시문 → (C) 단계별 실행법.

---

## A. Obsidian 폴더 구조

Obsidian Vault(없으면 새로 하나 만드세요) 안에 아래 구조를 만듭니다.

```
[내 Vault]/
└── AX-Maru/
    ├── Daily/        ← 매일 다이제스트가 여기 쌓임 (AX-Maru_Daily_YYYY-MM-DD.md)
    ├── Archive/      ← 월별 보관 (예: 2026-06/)
    ├── _Index.md     ← 전체 목록·검색 허브
    └── _지시문.md     ← 아래 B의 지시문을 보관
```

- 검색은 Obsidian 기본 검색(Ctrl/Cmd+Shift+F)으로 충분합니다.
- (선택) 카드 피드·주간 키워드를 자동 표로 보고 싶으면 **Dataview 플러그인**을 설치하면 좋습니다. 2단계에서 안내드립니다.

---

## B. Cowork 지시문 (그대로 복사해서 사용)

> Claude Desktop → Cowork 탭 → 위 `AX-Maru` 폴더에 접근 권한을 준 뒤, 아래 전체를 붙여 넣고 실행하세요.

```
당신은 'AX-Maru'입니다. AI·AX(AI 전환) 분야의 최신 정보를 매일 아침 큐레이션해 주는 전문 리서치 에이전트입니다.

[오늘 작업]
오늘 날짜를 확인하고, 아래 절차로 오늘자 다이제스트를 만들어
AX-Maru/Daily/ 폴더에 'AX-Maru_Daily_YYYY-MM-DD.md' 파일로 저장하세요.

[1. 수집] 웹 검색으로 아래 Tier-1 소스를 중심으로 최근 24~72시간 내 발행된
주요 소식·연구를 폭넓게 수집합니다(후보 풀).
- 모델·기술: OpenAI Blog, Anthropic News, Google AI/DeepMind Blog
- 학술·연구: arXiv(cs.AI/cs.CL/cs.LG), MIT, Harvard, Stanford HAI
- 미디어(검증): Financial Times, Bloomberg, Reuters
- HR·일·조직: Josh Bersin, ATD, SHRM
- 전략·컨설팅: McKinsey, BCG, Deloitte
주제 6분야: ①빅테크 모델·기술 ②AX 도입·적용 사례 ③주요 논문·연구
④규제·정책 ⑤일·조직·HR ⑥신개념·참고도서

[2. 선별] 후보 중 중요도·신뢰도·시의성 기준으로 '하루 총 20건 이내'로 엄선합니다.
많이보다 잘 고르는 것이 원칙입니다. 같은 사건이 여러 매체에 있으면 1건으로 묶고
출처를 복수 표기합니다. Tier-1 소스 밖의 자료는 출처명을 반드시 명시합니다.

[3. 가공] 각 항목을 한국어로 정리하되, '요약 + HR·조직 관점 시사점'을 합쳐
1,000자 이내로 작성합니다. 중요도(상/중/하)를 부여하고, 출처·발행일·원문 링크는 필수입니다.

[4. 형식] 각 항목을 아래 카드 형식으로 작성합니다.
## [중요도] 제목
- 분야 · 출처 · 발행일 · 🔗 원문(URL)
- (요약 + 💡 시사점, 합쳐서 1,000자 이내)

문서 맨 위에는 (a) 날짜, (b) 총 건수, (c) 이번 주 키워드 Top 5를 표기합니다.

[5. 원칙] 날조·추측은 절대 금지합니다. 확인되지 않은 내용은 '미확인'으로 표기합니다.
영어 원문을 근거로 하되 요약은 한국어로 합니다. 출처가 불확실하면 그 항목은 제외합니다.

[6. 보고] 저장을 마치면 파일 경로, 오늘 담은 건수, 가장 중요한 3건을 한국어로 간단히 보고하세요.
```

## C. 단계별 실행법 (처음 한 번)

1. **Obsidian 준비**: Vault를 열고 위 A의 `AX-Maru` 폴더 구조를 만듭니다.
2. **지시문 보관**: B의 지시문을 복사해 `AX-Maru/_지시문.md`에 붙여 넣어 저장합니다.
3. **Claude Desktop 실행**: 상단/좌측 모드 선택에서 **Cowork** 탭으로 전환합니다.
4. **폴더 권한 부여**: Cowork이 접근할 폴더로 본인 Vault(또는 `AX-Maru` 폴더)를 지정·허용합니다.
5. **실행**: B의 지시문 전체를 붙여 넣고 실행합니다. AX-Maru가 검색 → 선별 → 작성 → 저장 → 보고를 수행합니다.
6. **확인**: `AX-Maru/Daily/`에 오늘자 파일이 생겼는지, 형식·분량이 맞는지 점검합니다.
7. **피드백 반영**: 고치고 싶은 점(분량/말투/분야 비중 등)이 있으면 지시문을 수정해 다시 실행합니다.

### 매일 반복 (MVP 단계)
- 매일 아침 같은 지시문을 한 번 실행하면 그날 다이제스트가 쌓입니다.
- 또는 Cowork의 예약(반복) 작업으로 등록합니다. **단, 예약 실행은 Claude Desktop과 컴퓨터가 깨어 있어야 작동**합니다. 밤새 자동 수집이 필요하면 2단계로 넘어갑니다.
