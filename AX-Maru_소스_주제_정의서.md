# AX-Maru — 소스 · 주제 정의서 (v0.2 MVP)

> AX-Maru가 "무엇을, 어디서, 어떻게" 수집·가공할지 정의하는 설정 파일입니다.
> Cowork 지시문과 Obsidian 다이제스트가 모두 이 문서를 기준으로 작동합니다.
> v0.2 변경: 분량 = 하루 총 20건 이내 / 요약+시사점 합산 1,000자 이내.

---

## 1. 핵심 원칙

- **품질 > 속도.** 많이가 아니라 잘 고른다.
- **Tier-1 소스 우선.** 아래 신뢰 목록 밖의 출처는 보조로만, 출처명 표기.
- **영어 원문 수집 → 한국어 요약.** (2단계에서 원문/번역 토글 제공)
- **모든 항목에 출처·발행일·원문 링크 필수.**
- **날조·추측 금지.** 확인 안 된 내용은 '미확인' 표기.

|------|--------|

| 제공 시각 | 매일 아침 08:00 (KST) — Obsidian 다이제스트 |

| **요약 분량** | **요약 + HR·조직 시사점 합산 1,000자 이내** |

| 필수 메타 | 출처 · 발행일 · 원문 링크 |
| 중복 처리 | 유사 주제 자동 클러스터링(1건으로 묶고 출처 복수 표기) |

## 5. 출력 형식 (Obsidian 카드)

```
## [중요도] 제목
- 분야 · 출처 · 발행일 · 🔗 원문(URL)
- (요약 + 💡 시사점, 합쳐서 1,000자 이내)
```

페이지 상단: (a) 날짜, (b) 총 건수, (c) 주간 키워드 Top 5.

## 6. 단계 로드맵

- **1단계(MVP, 현재):** Cowork으로 매일 다이제스트 생성·검증. 컴퓨터를 켜둔 채 실행/예약.
- **2단계(자동화):** Claude Code 스크립트화 → 상시 환경에서 무인 실행 + 08시 대시보드 + 이메일 발송 + 원문/번역 토글.
